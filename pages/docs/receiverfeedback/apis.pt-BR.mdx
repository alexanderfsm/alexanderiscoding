# APIs

## Validar requisição para Desktop ou Mobile App

```js
fetch("https://examplename.vercel.app/api/shield", {
  method: "POST",
  headers: {
    "Content-Type": "application/json; charset=UTF-8",
    Authorization: "e632efe_access_token_5452b",
  },
});
```

### Parâmetros de resposta

- `200`: Retorna os códigos de autorização válido por 30 segundos
- `401`: Retorna a mensagem de permissão insuficiente
- `406`: Retorna a mensagem de dados insuficiente
- `500`: Retorna o erro ocorrido

## Validar requisição para Website

Primeira requisição

```js
fetch("https://examplename.vercel.app/api/shield", {
  method: "POST",
  headers: {
    "Content-Type": "application/json; charset=UTF-8",
  },
});
```

Próximas requisições

```js
fetch("https://examplename.vercel.app/api/shield", {
  method: "POST",
  headers: {
    "Content-Type": "application/json; charset=UTF-8",
  },
  body: JSON.stringify({
    tag: "1670254702349",
  }),
});
```

### Parâmetros para envio

- `tag`: Nome da tabela existente ou a ser criada

### Parâmetros de resposta

- `200`: Retorna os códigos de autorização válido por 30 segundos
- `401`: Retorna a mensagem de permissão insuficiente
- `406`: Retorna a mensagem de dados insuficiente
- `500`: Retorna o erro ocorrido

## URL

```
https://examplename.vercel.app.com/api/new
```

## Método de uso

```js
fetch("https://examplename.vercel.app.com/api/new", {
  method: "POST",
  headers: {
    "Content-Type": "application/json; charset=UTF-8",
  },
  body: JSON.stringify({
    level: 3,
    text: "feedback",
    source: "origin from receiver feedback (website, desktop app, mobile app)",
    version: "1.0",
  }),
});
```

## Parâmetros para envio

- `level`: Number | 1, 2, 3, 4 ou 5
- `text`: String
- `source`: String
- `version`: String
- `message`: String | Mensagem personalizada (Opcional)
- `application`: String (Opcional)

## Parâmetros de resposta

- `200`: Retorna o código do documento criado
- `401`: Retorna a mensagem de permissão insuficiente
- `406`: Retorna a mensagem de dados insuficiente
- `500`: Retorna o erro ocorrido
