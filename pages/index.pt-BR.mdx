# Introdução

Bem vindo ao Nextra! Este é um modelo básico para documentações. Você pode usá-lo como ponto de partida para seu próprio projeto :)

## O que é Nextra?

Uma estrutura de geração de sites **simples**, **poderosa** e **flexível** com tudo o que você ama do Next.js.

## Documentação

A documentação está disponível em [https://nextra.site](https://nextra.site).
